About Me

Lives in Stockholm, Sweden. Have been working with computers and software since the early 1990s. Initially using Basic, Quick Basic, and later Visual Basic (versions 3 through 6).

In 2004, started with the .NET Framework using both Visual Basic .NET and C#. Got tired of programming around 2007, and from 2013 focused mainly on PowerShell for setting up and managing SharePoint.

Recently found the passion for programming again with Python — and haven't looked back since.

Career

Had my own company in the early 1990s developing in Microsoft Access, and held classroom courses teaching MS-DOS, Windows, and MS Works.

Worked for Microsoft Sweden between 1995–2008 in various roles: Support Engineer, Technical Account Manager, and Consultant.

Since then, been working mostly at consultant companies as an Architect/IT Pro with SharePoint Server, Microsoft 365, and Azure.

Programming Languages

  • Python (primary focus)
  • PowerShell
  • C# / VB.NET (historical)
  • Visual Basic (historical)
