A list and a generator on the list

numbers = [1, 2, 3, 5, 7]
squares = (n**2 for n in numbers)

Use the generator once into a tuple

tuple(squares)

(1, 4, 9, 25, 49)

Use it a second time and nothing's there

sum(tuple(squares))

0

Same list and generator

numbers = [1, 2, 3, 5, 7]

Check in the number 9 is in the generator
